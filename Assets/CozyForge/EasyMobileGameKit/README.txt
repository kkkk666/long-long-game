GETTING STARTED
1. Click on “Import TMP Essentials” when you see the window
2. Now you can double click the “DEMO” scene inside CozyForge>EasyMobileGameKit>Scenes
3. Next you need to open the Project Settings window in Edit > Project Settings and go to the Services section
4. Select your organization
5. Use an existing cloud project or create a new fresh one
6. Now go to the Environments section under Services and select an environment
7. Now open the Deployment window from Services > Deployment
8. With the Deployment window open, make sure everything under EasyMobileGameKit is selected and click on Deploy All or Deploy Selected
9. Make sure everything has been deployed successfully (green). If anything is red, click on the red item and click on Deploy Selected
10. You are now ready to use Easy Mobile Game Kit! You can press Play and enter the demo scene. You should see the message “Authentication: SUCCESSFUL” in the console
